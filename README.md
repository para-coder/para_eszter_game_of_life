# para_eszter_game_of_life
Conway's Game of Life implemented in Python using Jupyter Notebook and Seaborn.
This project implements Conway's Game of Life in Python using:

- Jupyter Notebook
- NumPy
- Matplotlib
- Seaborn

## Features

- Random initial grid
- Conway's Game of Life rules
- Animated visualization
- Customizable grid size and speed

## Files

- `game_of_life.ipynb` – Main notebook
- `requirements.txt` – Required Python packages
- `images/screenshot.png` – Example output

## Installation

```bash
pip install -r requirements.txt
